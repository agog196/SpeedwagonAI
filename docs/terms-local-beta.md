# SpeedwagonAI Local Beta Terms

Last updated: 2026-06-05

These local beta terms are plain-language expectations for early testing.

## Local Beta Status

SpeedwagonAI is experimental software. It may have bugs, lose state, create noisy suggestions, fail to capture audio correctly, or require manual setup. Use it with data you are comfortable testing locally.

## Your Responsibilities

You are responsible for:

- keeping your Mac, repo checkout, and local data secure;
- managing your own API keys and provider accounts;
- reviewing all drafts, tasks, suggestions, and captures before acting on them;
- confirming consent before using meeting-bot capture;
- checking provider costs for OpenAI, Google, Recall.ai, or any future external service.

## No Automatic External Actions

SpeedwagonAI does not send emails automatically. Gmail behavior is draft-only.

SpeedwagonAI creates Calendar events only when you explicitly use the Calendar create flow. It does not edit/delete Calendar events or write Apple Reminders in this beta.

Notification taps open review surfaces and do not complete tasks, send email, create Gmail drafts, or perform other external actions automatically.

## External Services

If you configure external services, their own terms and privacy policies also apply. SpeedwagonAI passes selected content to those services only when you use the related feature.

## Data Export And Wipe

SpeedwagonAI provides local export and wipe tools for Speedwagon-owned data. You should inspect exports before sharing them. Wipe requires the exact confirmation phrase and is intended to remove configured local app data, not unrelated repo files.

## No Warranty

This beta is provided as-is for local testing. Do not rely on it as the sole source of truth for critical obligations, legal deadlines, medical decisions, financial decisions, or other high-stakes work.

## Future Terms

These beta terms should be replaced or reviewed before any public production release, signed distribution, hosted service, billing, or team/cloud sync.
